# Notifica App (demo version)

My original app for help my colleagues to punch the clock at the right time. Technologies and knowledge used:
* Swift,
* SwiftUI,
* Clean Code,
* UX/UI Design,
* Responsive Layout,
* MVVM Architectural Pattern,
* Local Notifications Alerts.
